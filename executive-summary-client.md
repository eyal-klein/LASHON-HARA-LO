# הצעה לפיתוח אתר V2
## עמותת "לשון הרע לא מדבר אליי"

**מוגש על ידי**: ניוקלאוס - מבית THRIVE SYSTEM  
**תאריך**: דצמבר 2024

---

## רקע

האתר הנוכחי של העמותה בנוי על פלטפורמת WordPress עם Elementor. למרות שהאתר משרת את העמותה, קיימים מספר אתגרים: מהירות טעינה איטית, קושי בתחזוקה ועדכון תוכן, עלויות תחזוקה גבוהות, ומגבלות בהתאמה לצרכים העתידיים של העמותה.

---

## הפתרון המוצע

אנו מציעים לבנות אתר חדש מהיסוד, תוך שמירה על **זהות המותג הקיימת** (לוגו, צבעים, תוכן), עם טכנולוגיה מודרנית שתספק:

### יתרונות מרכזיים

| יתרון | תיאור |
|-------|-------|
| **מהירות** | טעינה מהירה פי 5-10 מהאתר הנוכחי |
| **ניהול תוכן פשוט** | מערכת CMS ידידותית לעדכון תוכן ללא ידע טכני |
| **תרומות מאובטחות** | אינטגרציה עם Stripe לקבלת תרומות בכרטיס אשראי |
| **ניוזלטר אוטומטי** | שליחת עדכונים אוטומטיים למנויים |
| **גלריה דינמית** | ניהול קל של תמונות שגרירים ופעילויות |
| **רספונסיביות מלאה** | תצוגה מושלמת בכל מכשיר |
| **אבטחה מתקדמת** | הגנה מפני התקפות סייבר |

## מבנה האתר המוצע

### דפים ראשיים

1. **דף הבית** - Hero מרשים, הסבר על לשון הרע, ציטוט גנדי, טופס התחייבות, מסלולי שותפות
2. **אודות** - סיפור העמותה, חזון ומטרות
3. **הפעילות שלנו** - חלוקות, שגרירים, סדנאות, תערוכות, קמפיינים
4. **גלריה** - תמונות שגרירים, פעילויות ושותפים
5. **צור קשר** - טופס יצירת קשר
6. **תרומה** - מערכת תרומות מאובטחת
7. **חנות** - קישור לחנות המוצרים

### פאנל ניהול (Admin)

- ניהול תוכן האתר
- צפייה בהתחייבויות ומנויים
- ניהול גלריה ופעילויות
- צפייה בתרומות
- שליחת ניוזלטר

## תכונות עיקריות

### טופס התחייבות אישית
מבקרים יוכלו להתחייב להימנע מלשון הרע ולהירשם לעדכונים. הנתונים נשמרים במערכת ומאפשרים מעקב אחר מספר המתחייבים.

### מערכת שותפויות
ארבעה מסלולי שותפות עם טפסי הרשמה:
- שגרירים - מתנדבים להפצת המסר
- תמיכה כספית - תורמים קבועים
- בתי ספר - תוכניות חינוכיות
- השראה - שיתוף סיפורים אישיים

### מערכת תרומות (Stripe)
- תרומה חד-פעמית או חודשית
- תמיכה בשקלים, דולרים ואירו
- קבלה אוטומטית במייל
- פאנל ניהול לצפייה בתרומות

### ניוזלטר אוטומטי
- הרשמה פשוטה מהאתר
- שליחת עדכונים על פעילויות
- ניהול רשימת מנויים
- סטטיסטיקות פתיחה וקליקים

### מערכת CMS
- עריכת תוכן בקלות
- העלאת תמונות
- ניהול גלריה
- עדכון פעילויות

## טכנולוגיה

### תשתית ענן - Google Cloud Platform (GCP)

| רכיב | טכנולוגיה |
|------|-----------|
| **אחסון האפליקציה** | Cloud Run (Serverless) |
| **בסיס נתונים** | Cloud SQL (MySQL) |
| **אחסון תמונות** | Cloud Storage |
| **CDN** | Cloud CDN (מהירות גלובלית) |
| **אבטחה** | Cloud Armor (הגנה מפני התקפות) |
| **SSL** | תעודת אבטחה מנוהלת |

### יתרונות הטכנולוגיה

- **מדרגיות אוטומטית** - האתר מתרחב אוטומטית בעת עומס
- **זמינות גבוהה** - 99.9% uptime
- **אבטחה מובנית** - הגנה מפני DDoS, XSS, SQL Injection
- **גיבויים אוטומטיים** - שחזור מידע בכל עת
- **שרת בישראל** - מהירות מקסימלית לגולשים ישראלים
