### **למה בחרנו בטכנולוגיות האלה?**

#### **Frontend - React 19 + Tailwind 4**
**מה זה אומר בשפה פשוטה:**
- ממשק משתמש מהיר וחלק כמו אפליקציה
- עיצוב מודרני ועקבי
- נטען מהר גם באינטרנט איטי

**למה זה טוב:**
- React 19 = הגרסה החדשה ביותר (יצאה נובמבר 2024)
- Tailwind 4 = עיצוב מהיר ועקבי
- שפה אחת לכל האתר = קל לתחזוקה

#### **Backend - Node.js 22 + tRPC 11**
**מה זה אומר:**
- שרת מהיר ויציב
- תקשורת מאובטחת בין הדפדפן לשרת
- Type-safe = פחות באגים

**למה זה טוב:**
- Node.js 22 = הגרסה החדשה ביותר (אפריל 2024)
- tRPC = אין צורך ב-REST API מסורבל
- TypeScript = קוד מאובטח יותר

#### **Database - TiDB (MySQL Compatible)**
**מה זה אומר:**
- מסד נתונים ענק שגדל אוטומטית
- תמיכה במיליוני רשומות
- גיבויים אוטומטיים

**למה זה טוב:**
- MySQL compatible = קל למצוא מפתחים
- Serverless = משלמים רק על מה שמשתמשים
- Auto-scaling = לא צריך לדאוג על גדילה

#### **Hosting - Google Cloud Run**
**מה זה אומר:**
- האתר רץ על שרתי Google
- גדילה אוטומטית לפי עומס
- זמינות 99.95%

**למה זה טוב:**
