## 1. סקירת המערכת

### 1.1 מטרה

מערכת ניהול מקיפה המאפשרת לצוות העמותה לנהל את כל היבטי האתר ללא צורך בידע טכני, כולל תוכן, מוצרים, הזמנות, התחייבויות, תרומות ועוד.

### 1.2 הרשאות משתמשים

| תפקיד | הרשאות |
|-------|--------|
| **Super Admin** | גישה מלאה לכל המערכת |
| **Admin** | ניהול תוכן, מוצרים, הזמנות |
| **Editor** | עריכת תוכן, גלריה, פעילויות |
| **Viewer** | צפייה בדוחות בלבד |

### 1.3 מבנה תפריט Admin

```
📊 דשבורד
├── 🛒 מכירות
│   ├── מוצרים
│   ├── קטגוריות
│   ├── הזמנות
│   ├── קופונים
│   ├── מלאי
│   └── סיטונאות
├── 📝 תוכן
│   ├── דפים
│   ├── סקשנים
│   ├── תמונות
│   └── קבצים
├── 👥 קהילה

│   ├── מנויי ניוזלטר
│   ├── שותפויות
│   └── הודעות
├── 🖼️ מדיה
```
